Welcome to Wikipedia, a collaborative project to produce a complete encyclopedia from scratch. We started in january 2001 and already have over 19,000 articles. We want to make over 100,000, so let's get to work--anyone can edit any article--copyedit, expan an article, write a little, write alot. See the Wikipedia FAQ for information on how to edit pages and other questions.
